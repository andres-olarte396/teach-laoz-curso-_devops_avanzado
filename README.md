# teach-laoz-curso-_devops_avanzado

Este repositorio contiene el material del curso avanzado de DevOps de Teach LaoZ.

## Estructura del repositorio

- `cronograma.md`: Cronograma del curso.
- `DIAGRAMAS_TECNICOS.md`: Diagramas técnicos de referencia.
- `generate_all_audio.ps1`: Script para generar los audios de los módulos.
- `pensum_competencias.md`: Pensum y competencias del curso.
- `plan_curricular.md`: Plan curricular detallado.
- `media/`: Archivos de audio generados para cada módulo y subtema.
- `modulos/`: Contiene los módulos del curso, cada uno con sus temas, subtemas y materiales:
  - `*_contenido.md`: Contenido teórico del subtema.
  - `*_ejercicios.md`: Ejercicios prácticos.
  - `*_evaluacion.md`: Evaluaciones y pruebas.
  - `*_guion.md`: Guion para grabación de audio o video.

## ¿Cómo usar este repositorio?

1. Explora los módulos en la carpeta `modulos/` para acceder a los materiales de cada tema.
2. Utiliza el script `generate_all_audio.ps1` para generar los audios si es necesario.
3. Consulta los diagramas y cronogramas para planificar el avance del curso.
